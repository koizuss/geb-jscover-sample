Latest Release

The latest release is v0.3.0. This release contains some major changes including:

  • Function coverage had been added
    Thanks to Howard Abrams for his contributions. Function coverage is included by default, but can be turned off with the '--no-function' switch.
  • Branch coverage included by default
    The '--branch' switch has been replaced with '--no-branch' which turns off branch coverage data collection.

See the release notes for more information.
